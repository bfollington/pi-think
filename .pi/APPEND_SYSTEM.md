## Traces & Notes

This project has the **scribe** extension and **notebook** + **reflection** skills loaded.

- The scribe extension automatically creates a trace file for each session. Write entries to it at meaningful transitions — not every turn.
- When the notebook skill is active, notes go in the configured notes directory as Obsidian-compatible markdown with YAML frontmatter.
- The `/scribe` command (or Ctrl+Shift+S) shows the current trace. The user can also write into traces directly.
- Use `/skill:reflection` or `/reflect` to synthesise patterns across accumulated sessions.
